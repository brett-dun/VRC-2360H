"Public" Methods

void init()

This is the first method that must be run when using this library. This will configure the background tasks and variables. Once you run this method, do not continue to use motor[myMotor] = someSpeed; as this will have conflicting results with this library.

Example:

init();

Status: Development

Known Issues: None


void addSlave(tMotor master, tMotor slave)

A "slave" motor is a motor that follows what the "master" motor does. Use this method immediatly after calling init() to setup the relationship between motors. This motor takes in two paramaters both of the tMotor data type. tMotor just refers to the name of your motors and their corrosponding ports.

Example:

addSlave(leftDrive1, leftDrive2);

Status: Development

Known Issues: None
